Managing cameras

Installing a HTTP camera

To install a HTTP camera go to Install > Install camera in the left navigation bar to open the setup wizard.

1 - Enter a name and description for the new camera and click Next

2 - Select an interface. For a HTTP camera, choose HTTP and click Next

3 - In the next step, enter the URL where the camera provides a current image. Optionally, set a watchdog interval to receive notifications if the camera fails to communicate within a specified time.

5 - Finally, click Finish. If successful, you’ll be redirected to the cameras page, where installed cameras are displayed. From this page, you can also disable the watchdog or delete the camera.
